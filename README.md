## Getting started

Inside Command line:

` mysql -u root -p mysql> CREATE DATABASE testDB; mysql> USE testDB;` <br />
` mysql> CREATE TABLE student (uid DECIMAL(3, 0) PRIMARY KEY, name VARCHAR(30), score DECIMAL(3, 2));` <br />
` mysql> INSERT INTO student VALUES(1, ’x, 0.1); mysql> INSERT INTO student VALUES(2, ’y’, 0.4);` <br />
` mysql> SELECT * FROM student;`
